Best-effort scan The orphan flag is a heuristic, not proof. It matches each image filename against the site's HTML, JS, CSS backgrounds, and the data registries (articles/articles-data.js, js/featured-yachts.js). It will NOT catch images referenced by a name built at runtime, pulled from an external CMS, or used only in files outside this repo. Treat "likely unused" as "verify before deleting," never as an automatic delete list.
