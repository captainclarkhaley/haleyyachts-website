Email Composer

How to use:
(1) Upload a hero image (recommended 1200x630 JPEG, <300 KB). It saves into /images/email/hero/ with a timestamped filename.
(2) Optional: add an H2 headline above the body.
(3) Write or paste your body content. Basic formatting buttons cover most needs.
(4) The preview pane shows what Constant Contact will render. Refresh it any time.
(5) Click Copy HTML, then paste into Constant Contact's Custom Code template editor. The subject line goes into CC's subject field separately.

CC handles all tracking (opens, clicks, unsubscribe, list management). No tracking code is added on our side.

With a GitHub token set below, the hero image commits straight to GitHub from any browser. With no token, saving falls back to picking the local haleyyachts-website folder on this machine. The composed email HTML is always copy/pasted into Constant Contact, regardless of token.
