HALEY YACHTS ADMINISTRATION

Internal tools for maintaining site content

Security Notice Password protection is configured via /admin/.htaccess but must be activated on the GoDaddy server. Go to cPanel > Directory Privacy, select /public_html/admin, check "Password protect this directory," and create a user. Until that is done, this area is unprotected if uploaded to the web.

Content Tools

Manage up to six featured yachts shown on the Buy page. Browse an image from your computer, it copies into the right folder on save. Empty slots are hidden from the site.

Upload Word docs and images, select article type, and publish directly to the site. No Git backend required.

Site Analytics Live

Pre-filled deep links into Google Analytics 4 and Microsoft Clarity. Real-time visitors, session recordings, heatmaps, conversion events.
