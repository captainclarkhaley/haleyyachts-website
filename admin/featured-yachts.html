Featured Image Maintenance

How to use:
(1) Copy the image you want to display to the following folder: /images/yachts/featured
    IMPORTANT: File names must be all lowercase and contain no spaces. Use hyphens to separate words (e.g., riviera-465suv.jpg, southern-wind.jpg). The web server is case-sensitive and spaces can break links.
(2) In the Featured Image Position, enter which featured yacht you want to edit (1, 2, or 3).
(3) Enter the Featured Image Name (the filename you saved in the featured folder, including extension).
(4) Enter the Featured Image Title (max 60 characters).
(5) Enter the Featured Image Description (max 250 characters).
(6) Click SAVE. The first time you save, your browser will ask you to locate /js/featured-yachts.js - select it once and all future saves will write to that file automatically.
