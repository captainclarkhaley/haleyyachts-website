Featured Yacht Maintenance

How to use:
(1) Pick the Position to edit, then click Load Current Values.
(2) Click Browse and select an image from your computer. It copies into /images/yachts/featured/ when you save. The Image Filename box fills itself in and is read-only - it is only the name the file is stored under, not anything a visitor sees. To reuse a picture already on the site, use Use a different image already on the site.
(3) Enter the Featured Yacht Title - Year Make Length Model Price, e.g. “2023 Riviera 64 Sports Motor Yacht $3,539,000” (max 60 characters) - and the Featured Yacht Description (max 250 characters).
(4) Optional: attach a Full Details PDF. Browse to a PDF (it copies into /documents/yachts/ using the file’s original filename) or paste an external spec-sheet URL. Leave blank to hide the Full Details button on the card.
(5) Click SAVE.

Reorder slots: Load a position, choose a destination from Move to Position, then click MOVE. The boat you move takes the new slot and the other slots shift to make room.

With a GitHub token set below, saving commits straight to GitHub from any browser. With no token, saving falls back to picking the local haleyyachts-website folder on this machine. Either way, Clark runs the cPanel pull to deploy.
