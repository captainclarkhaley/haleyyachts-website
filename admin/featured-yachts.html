Featured Yacht Maintenance

How to use:
(1) Pick the Position (1-6) to edit, then click Load Current Values.
(2) Click Browse and select an image from your computer. It will be copied into /images/yachts/featured/ when you save, and the filename will be auto-cleaned (lowercase, hyphens, no spaces). You can also type a filename manually if the image is already in the folder.
(3) Enter the Title (max 60 characters) and Description (max 250 characters).
(4) Optional: attach a Full Details PDF. Browse to a PDF (it copies into /documents/yachts/featured/ using the file's original filename) or paste an external spec-sheet URL. Leave blank to hide the Full Details button on the card.
(5) Click SAVE.

Reorder slots: Load a position, choose a destination from Move to Position, then click MOVE. The boat you move takes the new slot and the other slots shift to make room.

With a GitHub token set below, saving commits straight to GitHub from any browser. With no token, saving falls back to picking the local haleyyachts-website folder on this machine. Either way, Clark runs the cPanel pull to deploy.
