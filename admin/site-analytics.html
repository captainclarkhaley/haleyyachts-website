SITE ANALYTICS

Deep links into Google Analytics 4 (GA4) and Microsoft Clarity. Both are wired into every public page via js/analytics.js. GA4 covers metrics and conversions; Clarity covers session recordings and heatmaps.

Find it in GA4 under Admin → Property Settings → Property ID. Saved locally in this browser only. Without it, links land on whatever GA4 property you viewed last (usually fine if you're signed in).

Live Visitors & Engagement

GA4

Real-time visitors

Who is on the site right now, what page, where from. Use this to verify tracking is firing.

 GA4

Engagement & time on page

Average engagement time per page, which pages keep people, which don't.

 Clarity

Clarity dashboard

Sessions, dead clicks, rage clicks, scroll depth, JS errors at a glance.

Visitor Behavior

Clarity

Session recordings

Watch real visitors navigate the site. Patrick recommends sampling 5-10 a month to inform listing copy and gallery layout.

 Clarity

Heatmaps

Click and scroll heatmaps per page. Especially useful for the Buy and Featured Yachts pages once traffic builds.

 GA4

Pages & screens report

Page views ranked. See which articles, which Riviera models, and which marketing pages get attention.

Conversion Events & Lead Signals

GA4

Events report

All tracked events: phone clicks, email clicks, contact submissions, valuation requests, engaged buyers.

 GA4

Conversions report

Only events you've marked as "key events" in Admin → Events. Use this to track lead-quality signals.

 GA4

Traffic sources

Where visitors come from: Google, direct, social, referrals. Pair with Patrick's marketing campaigns to measure ROI.
